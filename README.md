# Palindrome Checker using Stack (C++)

This project is a **simple Palindrome Checker** implemented using **C++**, **Stack (STL)**, and **Object-Oriented Programming (OOP)** concepts.  
It checks whether a given word reads the same **forward and backward**.

## ğŸ§¾ Program Description

- The program defines a **PalindromeChecker** class
- All characters of the input string are pushed into a stack
- The string is traversed again and compared with stack elements
- If all characters match, the string is declared a **Palindrome**
- Otherwise, it is **Not Palindrome**
